Uso de knitr con LaTeX

Introducción

El libro knitr permite construir un archivo LaTeX que integra resultados de R. Evidentemente, el libro con todas sus dependencias debe estar en la biblioteca, en caso contrario utilice la orden install.packages("knitr", dependencies = TRUE) .

Una vez cargado con la orden library(knitr), basta con crear un archivo con la extensión Rnw que incluya un texto en formato LaTeX y además incluya unos bloques especiales que serán ejecutados en R y cuyos resultados, mezclados con el texto original, constituirán un archivo con el mismo nombre que el archivo suministrado y con extensión tex.

Esto es, si el archivo se llama 1.Rnw y se encuentra en el directorio de trabajo, se ejecuta la orden knit("1.Rnw") y se obtendrá el archivo 1.tex.

Los bloques especiales se identifican entre las órdenes especiales

<<>>= Indica que comienza el bloque de órdenes R
@ Indica que termina el bloque de órdenes R

Entre << y >> se pueden introducir metaórdenes para R, que afectan al tipo de salida de R.

Lo recomendable es escribir primero el documento LaTeX y compilarlo para que no tenga errores. Luego se añaden los bloques de órdenes de R (también previamente comprobadas) y continuamos el procedimiento.

En el archivo Rnw, si se va a utilizar texto español desde Windows, se debería incluir lo siguiente: 

\documentclass[10pt,a4paper,oneside]{book}
\usepackage[spanish]{babel}

%\usepackage[cp850]{inputenc}            
%Para que utilice los caracteres especiales de MS-DOS

\usepackage[ansinew]{inputenc}           
%Para que utilice los caracteres especiales de Windows
