Ejemplo de uso de knitr con HTML

Introducción

El libro knitr permite construir un archivo HTML que integra resultados de R. Evidentemente, el libro con todas sus dependencias debe estar en la biblioteca, en caso contrario debe usar la orden install.packages("knitr", dependencies = TRUE). Una vez cargado con la orden library(knitr), basta con crear un archivo con la extensión Rhtml que incluya un texto en formato HTML y además incluya unos comentarios especiales que serán ejecutados en R y cuyos resultados, mezclados con el texto original, constituirán un archivo con el mismo nombre que el archivo suministrado y con extensión html.

Esto es, si el archivo se llama 1.Rhtml y se encuentra en el directorio de trabajo, se ejecuta la orden knit("1.Rhtml") y se obtendrá el archivo 1.html (y varios archivos adicionales en el directorio figure si se incluyen gráficos).

Los comentarios especiales son de la forma 

<!--begin.rcode [metaórdenes]
	Órdenes de R
end.rcode-->
dónde las metaórdenes afectan al tipo de salida de R.

Ejemplos

Generación de 20 números de la normal 0, 1. 

    ## Esta línea es un comentario que no afecta al resultado final
    x=rnorm(20)
Si no quiere que aparezcan las órdenes que producen un resultado, basta con que use la metaorden echo=FALSE. En el siguiente bloque se presentan los valores anteriormente generados.

Este es el histograma de esos datos (centrado mediante la opción fig.align='center' y de tamaño 4x4 mediante las opciones fig.width=4, fig.height=4):

        ## @knitr histograma
    hist(x)
Si para hacer un gráfico se dan varias órdenes, el programa graba todas las imágenes intermedias y puede presentarlas todas. Si solo quiere mostrar alguna de ellas utilice la opción fig.keep. El valor predeterminado es fig.keep='high' que muestra solamente la última. Si hay demasiados gráficos y solo quiere conservar el último puede usar la opción fig.keep='last'.

Las expresiones de R se formatean adecuadamente de modo predeterminado. Si las órdenes son: 

		for(k in 1:10){fj=cos(
		sin(k)*
	k^
	2)+3;cat(k,"\t",fj,"\n")}
obtendrá: 
        for(k in 1:10){fj=cos(
                sin(k)*
        k^
        2)+3;cat(k,"\t",fj,"\n")}
